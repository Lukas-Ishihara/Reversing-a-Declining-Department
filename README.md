# Reversing a Declining Department

## Summary
Analyzed a $1.6M revenue decline using SQL to identify structural root causes and evaluate turnaround strategies. By modeling user behavior and profit margins, I provided the data-driven justification to pivot toward a sustainable engagement model.

## Key Highlights
**Problem:**  
- A sharp revenue drop led leadership to propose a high-cost cashback program without data on its potential ROI or ability to fix underlying churn.

**The Solution:**  
- Performed a diagnostic analysis using linear regression and ROI simulations to segment "whale" users and define a 8.3% profit margin floor.

**The Result:**  
- Proved 99% of users would not meet incentive thresholds, saving the department from a failed investment and adjusting to a low-cost VIP status system.
